# Project Agap Web Command Center

Project Agap Web is the command and oversight experience of the platform. It gives barangay leaders and responders a wider operational view of what is happening across the community during an emergency.

Back to the platform overview: [Project Agap](../../README.md)  
See the mobile field app: [Project Agap Mobile](../native/README.md)

## Purpose

The web command center is designed for the moments when response leaders need to move from individual reports to barangay-wide action.

It brings together the information that matters most during active response:

- current help requests
- accountability and household status
- communication delivery
- registry and welfare coordination
- map-backed operational awareness

## Command Experience

Project Agap Web acts as a high-level operations layer for emergency coordination.

- live dashboard for response KPIs
- unresolved `Need Help` queue for rapid action
- household registry visibility and updates
- emergency broadcast management
- SMS monitoring and delivery context
- oversight of barangay activity from a single command surface

## Communication And Response

The web experience is especially important when officials need to coordinate communication at scale.

- send barangay-wide or purok-targeted broadcasts
- monitor outbound communication activity
- track response-related communication workflows
- support operational follow-up from one screen

## Command-Level Visibility

While the mobile app is built for field action, the web app is built for command clarity.

- broader situational awareness
- easier scanning of multiple households and active needs
- stronger visibility into response queues
- decision support for barangay leadership and coordination teams

## Map-Aware Operations

The platform’s map capability is not limited to navigation. On the web side, it supports wider operational awareness for the barangay, helping response teams understand geography, evacuation context, and the larger picture behind field activity.

## Why The Web Experience Matters

Project Agap becomes much stronger when the mobile and web experiences work together:

- residents and responders generate information in the field
- officials and coordinators interpret and act on that information at command level
- both sides stay aligned through one shared emergency platform

This makes the web command center an essential part of the product, not just an admin panel.
